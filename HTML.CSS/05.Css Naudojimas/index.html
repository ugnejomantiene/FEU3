Vaikinis elementas, gali turėti TIK 1 tėvinį elementą.

Paveldėjimas

Paveldėti stilius gali TIK VAIKINIS ELEMENTAS IŠ TĖVINIO ELEMENTO.
Paveldimi stiliai yra tie, kurie keičia elemento išvaizdą, o ne būseną. Pvz.: color, font-size, font-family...
Nepaveldimi stiliai yra tie, kurie keičia elemento būseną. Pvz.: width, height, display, border, position...
Net paveldėti stiliai gali nebūti atvaizduoti dėl stilių prioritetų.

Numatytieji stiliai

Numatytieji stiliai yra automatiškai naršyklės suteikti stiliai. Tarp skirtingų naršyklių numatytieji stiliai gali minimalistiškai keistis ir šitaip sugadinti svetainės vaizdą vartotojams.
Tam, kad nebūtų problemų kuriant dizainą skirtingoms naršyklėms yra naudojami Reset arba Normalize CSSai.
Reset - resetina visus stilius.
Normalize - suvienodina visus stilius tarp naršyklių ir panaikina dalį kenkėjiškų stilių.

  2. ID - 100

  4. Element - 1
  5. CSS Points
  6. CSS Order (eiliškumas kode)
  7. User agents Styles (Default)
