Git - kodo versijavimo programa.

Github - svetainė, kurioje keliamos kodo versijos.

  1. Įsirašyti Git`ą
  2. Patikrinti ar gitas įrašytas terminale parašius "git --version"
  3. Susikurti gitHub paskyrą
  4. configinti gita
    1. git config --global user.name ugnejomantiene
    2. git config --global user.email upetkeviciute@gmail.com
    3. git config --global core.editor code
    4. git config --global core.autocrlf true
  5. git config --global -e
  6. source control dalyje paspausti publish to github

CSS framework - kažkieno kito parašytas CSS kodas, kurį tu gali naudoti.
Norint naudoti kažkieno frameworką reikia jį prijungti prie savo kodo CDN.
Dauguma frameworkų veikia suteikiant jų aprašytas klases kažkokiems html elementams.
Yra CLASSLESS CSS frameworkų. Juos naudojant nereikia suteikti klasių jokiems htmlo elementams, viskas aprašyta naudojant tagų selektorius.
