Transform

  • scale - keičia elemento dydį (default 1)
  • scaleX(x), scaleY(y), scaleZ(z), scale(x,y), scale3d(x,y,z)
  • rotate - pasuka elementą (deg)
  • rotateX(x), rotateY(y), rotateZ(z), rotate(z), rotate3d(x,y,z,a)
  • translate - pastumti elementą
  • translateX(x), translateY(y), translateZ(z), translate(x,y), translate3d(x,y,z)
  • skew - tempia elementą už kampų (deg)
  • skewX(x), skewY(y), skew(x,y)
  • matrix, matrix3d
  • perspective

Animacija

  • animation-timing-function - kaip animacijai paskirtys turimą laiką
  • animation-delay - po kiek laiko prasidės animacija
  • animation-iteration-count - kiek sykių vyks animacija
  • animation-direction - animacijos kryptis
  • animation-fill-mode - animacijos pradžios/pabaigos nustatymai
  • animation-play-state - animacijos būsena
  • animation-name - animacijos pavadinimas
