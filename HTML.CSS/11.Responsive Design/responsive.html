Mobile First

  • Iš mažesnio ekrano dydžio yra lengvesnis perėjimas į didesnį.Lengviau oridėti turinį negu sumažinti esamą
  • Koncentracija į turinį dėl mažo vietos kiekio
  • Telefonai naudojami plačiau. (Orientuojamės į didesnį vartotojų kiekį)
  • Google paieškos algoritmas geriau įvertina Mobile First dizainu sukurtas svetaines.

Naudojant Mobile First dizaino principą pradedame nuo mažiausio ekrano ir judame link didesnių.
