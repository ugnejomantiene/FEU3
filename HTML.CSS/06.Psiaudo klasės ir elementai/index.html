Pseudo Selektoriai

  • *
  • element
  • .class
  • #id
  • selector, selector
  • selector selector
  • selector > selector
  • element.class#id - elementas turintis nurodytą klasą arba klases ir/arba nurodytą ID
  • selector1 + selector2 - iš karto už selector1 einantis selector2
  • selector1 ~ selector2 - už selector1 visi einantys selector2 (būtinai tame pačiame lygyje)
  • [attribute] - elementas parenka būtent šitą atributą
  • [atrribute=value] - atributas turintis nurodytą reikšmę
  • [atrribute~=value] - atributas, kurio reikšmėje yra nurodytas žodis
  • [atrribute*=value] - atributas, kurio reikšmėje yra nurodyta žodžio dalis
  • [atribute|=value] - atributas, kurio reikšmė yra value arba value-...
  • [atribute^=value] - atributas, kurio reikšmė prasideda value...
  • [atribute$=value] - atributas, kurio reikšmė pasibaigia ...value
  • :link - paprasta/nelankyta nuoroda
  • :visited - lankyta nuoroda
  • :hover - užėjęs kursorius
    • ,
  • :active - aktyvi (paspaudimo momento) nuoroda
  • element:empty - tuščias elementas
  • ::marker -sąrašo elementų žymės
  • :not(p) - parenka viską, apart to, kas yra apskliausta
  • :first-child - parenka pirmą vaiką
  • :last-child - parenka paskutinį
  • :only-child - parenka vienintelį vaikinį elementą, jeigu jis yra vienas
  • :first-of-type - parenka pirmąjį nurodyto tipo elementą
  • :last-of-type - parenka paskutinį nurodyto tipo elementą
  • :only-of-type - parenka vienintelį nurodyto tipo elementą
  • :nth-child(n) - parenka nurodytus vaikinius elementus
  • :nth-last-child(n) - nuo galo parenka nurodytus vaikinius elementus
  • :nth-of-type(n) - renka ELEMENTUS pagal tipą (nebūtinai vaikinius)
  • :nth-last-of-type(n) - renka ELEMENTUS pagal tipą nuo galo
  • ::after - sukuria turinį už elemento
  • ::before - sukuria turinį prieš elementą
  • input:checked - pažymėtus ar parinktus radio/checkbox/option pasirinkimus
  • input:default - iš anksto parinktas pasirinkimas
  • input:enabled - inputas, kuris neturi disabled atributo
  • input:disabled - inputas, kuris turi disabled atributą
  • input:focus - inputas, ant kurio yra paspausta/fokusuotas
  • input:in-range - inputas, kurio vertė yra nurodytose ribose
  • input:out-of-range - inputas, kurio vertė yra už nurodytų ribų
  • input:indeterminate - inputas, kurio reikšmė yra neapibrėžta
  • input:invalid - inputas, kurio vertė yra netinkama
  • input:valid - inputas, kurio vertė yra tinkama
  • input:required - inputas su required atributu
  • input:optional - inputas be required atributo
  • input:read-only - inputas, su readonly atributu
  • input:read-write - inputas, be read only atributo
  • ::placeholder - pasako, kad stilizuosime placeholder įvesties laukelį
  • ::first-letter - parenka elemento pirmąją raidė
  • ::first-line - parenka elemento pirmąją eilutę
  • ::selection - parenka vartotojo pažymėtą informaciją/turinį
