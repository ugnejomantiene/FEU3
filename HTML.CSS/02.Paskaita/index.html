Stilizavimo tagai

  • b - bold - Pastorina tekstą
  • strong - Pastorina tekstą (turi semantinę reikšmę)
  • i - italic - Suteikia tekstui pasvirimo stilių
  • em - emphasis - Suteikia tekstui pasvirimo stilių (turi semantinę reikšmę)
  • sup - super - Pakelia tekstą į viršų, kaip laipsnį
  • sub - subscribt - Nuleidžia tekstą, kaip matematinį
  • span - Išskiria elemento dalį ar elementą

Formos

Tagai

  • form - Forma
  • input - Įvestis
  • select - Pasirinkimas iš variantų
  • option - Pasirinkimo variantai
  • textarea - Didelio teksto kiekio įvestis
  • label - Etiketė (tarkim įveskite info)
  • optgroup - Pasirinkimo viariantų grupė
  • button - Mygtukas
  • fieldset - Formos laukelis
  • legend - Formos laukelio pavadinimas

Atributai

  • type - Input tago tipas
  • for - Nurodo, kuriam įvesties/parinkties laukui yra priskirta etiketė
  • id - Unikalu spavadinimas į kurį nurodo FOR atributas
  • name - Įvesties lauko vardas, kuris bus perduotas serveriui su įvesties reikšme
  • cols - Stulpelių kiekis text area laukelyje
  • rows - eilučių kiekis text area laukelyje
  • placeholder - Užlaiko vietą, nurodo, ką reikia rašyti
  • value - Įvesties reikšmė
  • optgroup - grupės pavadinimas
  • label - Priskiriai colgroupui pavadinima
  • selected - Numatytasis select tago viduje esantis optionas
  • disabled - Neleidžia rinktis
  • minlength - Trumpiausia galima įvestis
  • minlength - Ilgiausia galima įvestis
  • required - Privaloma sąlyga
  • min - mažiausia leidžiama įvestis
  • max - didžiausia leidžiama įvestis
  • step - žingsnis, kuriuo didėja/ mažėja skaičius
  • paterrn - Reikalauja...
  • radio - Gali pažymėti tik vieną fiksuotą atsakymą
  • checkbox - Gali pažymėti kelis (kiek nurodyta) variantų. Turi turėt vienodus name, bet skirtingą value
  • checked - Numatytasis pasirinkimas
