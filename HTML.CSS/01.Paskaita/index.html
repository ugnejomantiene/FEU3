Tag sąrašas

  • p - paragraph - Paragrafas/ pastraipa skirta didesniam kiekiui tekstui.
  • h1-h6 - Heading 1/6 - Antraštės skirtos rašyti pavadinimamas ir tarpusavyje skiriasi teksto dydžiu
  • img - Image - Paveiksliukas su šaltiniu
  • a - Hypertext Anchor (link)- nuoroda
  • hr - Thematic Break (horizontal line)- eilutės lūžis su linija
  • br - Line Break - eilutės lūžis
  • ul - Unordered List - Nerikiuotas
  • ol - Ordered List - Rikiuotas
  • meta - Skirtas nurodyti puslapio meta informaciją (Meta duomenis)
  • title - Skirtas nurodyti puslapio tago pavadinimą

Atributų sąrašas

  • lang - Language - nurodo, kokia kalba vyrauja tage (naudinga automatiniams skaitytuvams)
  • charset - Character set - simbolių rinkinys, nurodo, kokią biblioteką naudoti
  • src - Source - šaltinis - nurodo failą
  • alt - Alternative Text - Alternatyvus tekstas (kai neužkrauna paveikslėlio)
  • width - Nurodo elemento plotį
  • height - Nurodo elemento aukštį
  • href - Hyper link reference - Nuoroda, nurodanti dažniausiai į puslapį (arba failą) -
  • target - Nurodo, kur atidaryti naują nuorodą

  • CTRL S - Išsaugoja
  • CTRL C - Kopijuoja
  • CTRL X - Iškerpa

  • CTRL F - Atidaro paieškos sistemą

  • SHIFT TAB - Tabuliuoja pažymėtą kodą į kairę
  • CTRL A - Pažymi viską
  • CTRL / - Komentuoja pažymėtas eilutes
  • CTRL Y - Atšaukia CTRL Z veiksmą
  • ALT Z - Word Wrap
  • CTRL R - Leidžia pasirinkti kažkokį failą
  • CTRL SHIFT L - Pažymi visus vienodus žodžius/sakinius, taip pat galima juos keisti
  • CTRL SHIFT K - Ištrina pažymėta/as eilutes
  • CTRL SHIFT J/I - Atidaro Developer Tools naršyklėje
  • CTRL D - Pažymi parinktą elementą esantį kode
